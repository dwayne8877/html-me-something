How to Install Kodi on an Amazon Firestick

STEP 1: Enter your Amazon account information

STEP 2:Settings> System> Developer Options

STEP 3: ADB Debugging> Off Apps from unknown> On

STEP 4: Applications> Collect app usage data> Off

STEP 5: Search ES File Explorer> Download > Open

STEP 6: +New> http://kodi.tv

STEP 7: Downloads> ARM> Open file> Install> Open

STEP 8: Sytem> File Manager> Add source> http://fusion.tvaddons.ag > Name the file “fusion”

STEP 9: System> Settings> Addons> Install from zip> start-here> hub wizard

STEP 10: Programs> Program Add-ons>Config Wizard> Android
